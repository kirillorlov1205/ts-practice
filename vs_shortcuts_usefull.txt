**VS shortcuts usefull**

- Erase line: Ctrl + X
- New line bellow = Ctrl + Enter
- New line above = Ctrl + Shift + Enter
- Rename: F2
- Show Usage: Shift + F12
- Move code Up/Down: Alt + Arrow Up/Down
- Dublicate selected area: Shift + Alt + Arrow down
- Select context: Shift + Arrow right
- Surround select area: Ctrl + K
- Fold all: Ctrl + K + 0
- Unfold all: Ctrl + K + J
